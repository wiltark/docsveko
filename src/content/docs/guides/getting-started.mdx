## 🚀 Installation

### Installation globale (CLI)

```bash
npm install -g veko
```

### Installation dans un projet

```bash
npm install veko
```

## 🛠️ Créer un nouveau projet

1. **Initialisation du projet**

   ```bash
   # Créer un nouveau projet
   veko setup mon-app

   # Avec options
   veko setup --name mon-blog --template blog --git

   # Templates disponibles : default, api, blog, admin
   ```

2. **Structure du projet**

   ```
   mon-app/
   ├── routes/          # Gestion des routes
   ├── views/           # Templates EJS
   ├── public/          # Fichiers statiques
   ├── plugins/         # Plugins personnalisés
   ├── config/          # Configuration
   └── package.json
   ```

3. **Application de base**

   ```javascript
   // app.js
   const { App } = require('veko');

   const app = new App({
     port: 3000,
     viewsDir: 'views',
     staticDir: 'public',
     routesDir: 'routes',
     layouts: {
       enabled: true,
       defaultLayout: 'main'
     },
     plugins: {
       enabled: true,
       autoLoad: true,
       pluginsDir: 'plugins'
     }
   });

   app.loadRoutes().listen();
   ```

4. **Démarrer le serveur**

   ```bash
   # Mode développement
   veko dev

   # Mode production
   veko start
   ```

## 💻 Exemple de route

```javascript
// routes/index.js
module.exports = {
  get: (req, res) => {
    res.render('index', { 
      title: 'Bienvenue sur Veko.js',
      message: 'Framework ultra-moderne pour Node.js'
    });
  }
};
```

## ⚡ Commandes essentielles

```bash
# Démarrer le serveur de développement
veko dev

# Créer un nouveau projet
veko setup mon-app

# Vérifier les mises à jour
veko update check

# Installer une mise à jour
veko update update

# Afficher l'aide
veko --help
```
