## Pourquoi Veko.js ?

Veko.js est un framework **ultra-moderne** et **intelligent** pour Node.js avec Express et EJS, conçu pour un développement rapide et efficace.

## Commandes rapides

```bash
# Démarrer le serveur de développement
veko dev

# Vérifier les mises à jour
veko update check

# Installer une mise à jour
veko update update

# Configurer l'auto-updater
veko update config
